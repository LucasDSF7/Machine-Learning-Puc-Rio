# PUC-Rio: Data Science & Analytics Sprint

This repository contains the coursework, notebooks, and datasets for the **Data Science and Analytics** postgraduate sprint at **PUC-Rio**.

The content is organized into three core disciplines, each focusing on different aspects of modern data science and AI.

Disciplines:

## Machine Learning

    Focus: Supervised learning fundamentals, including classification and regression tasks, along with advanced feature engineering.

## Analytics – Descriptive and Predictive

    Focus: Introduction to neural networks and deep learning, advanced neural architectures, and unsupervised learning methods.

## Advanced Analytics

    Focus: Neural network frameworks, computer vision applications, and Natural Language Processing (NLP).

## Repository Structure

Each module is contained within its own directory, including the relevant code and supporting datasets.
```
├── machine-learning/                  # Notebooks & data for supervised learning
├── analytics-descriptive-predictive/  # Notebooks & data for NN and deep learning
├── advanced-analytics/                # Notebooks & data for CV and NLP
└── README.md
```
